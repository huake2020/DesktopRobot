# USART通讯协议
Byte|功能|值
-|-|-|
0       |帧头0|0xcc
1       |帧头1|0xcc
2       |保留位|0
3       |保留位|0
4       |保留位|0
5       |数据长度|0~255
6       |LED状态|0：开   1：关
7       |左轮速度|0~255
8       |右轮速度|0~255
7       |左轮方向|0：前   1：后
9       |右轮方向|0：前   1：后
10~13   |左轮编码器单位时间采样值|0~65535 
14~17   |右轮编码器单位时间采样值|0~65535
18      |左轮编码器方向|0：正   1：反 
19      |右轮编码器方向|0：正   1：反
20      |生命周期|0~255
21~22   |校验值|0~65536
